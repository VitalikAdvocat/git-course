Hello Git!!!

Для просмотра доступных в git команд используют команду git --help

команда git init - инициализирует пустой репозиторий

С помощюю команды commit -m "name commit" мы делаем точку для возврата в подальшом к файлам своего рода якорь."

С помощбю команды cls очищаем десктоп терминала

С помощью команды add . добавляем за файлами которые отслеживаються изменения

При добавлении новых файлов в наш проект git автоматическы будет просить нас заними следить и выводить об етом ынфрмацыю . Если за изменениями в даном файле не нужно знать другим учасникам проекта то его добавляют в файл игнора с помощюю того что создаеться системный git файл .gitignore в котором вводиться названия файла , или папки но после /name folder
