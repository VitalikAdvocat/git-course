Hello Git!!!

Для просмотра доступных в git команд используют команду git --help

команда git init - инициализирует пустой репозиторий

С помощюю команды commit -m "name commit" мы делаем точку для возврата в подальшом к файлам своего рода якорь."

С помощбю команды cls очищаем десктоп терминала

С помощью команды add . добавляем за файлами которые отслеживаються изменения

При добавлении новых файлов в наш проект git автоматическы будет просить нас заними следить и выводить об етом ынфрмацыю . Если за изменениями в даном файле не нужно знать другим учасникам проекта то его добавляют в файл игнора с помощюю того что создаеться системный git файл .gitignore в котором вводиться названия файла , или папки но после /name folder. Так как в нашу папку был добаллен важный файл gitignore сделаем commit и запишем точку сохранения Для просмотра текущей ветки разработчика нужна команда git branch в нашем случаи *master

Для добавления нового проекта для другого разработчика создаеться новая ветка проекта с помощью команды git branch testuser //где testuser название новой ветки Для удаления ветки используеться соответствено команда git branch -D testuser Для перехода на созданую ветку используют команду git checkout testuser Можна также создать и автоматически туда перейти с помощью команды git checkout -b test // и мы перешли на ветку test
